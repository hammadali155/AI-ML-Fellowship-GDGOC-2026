# Modules Directory

This directory contains standalone Python modules that demonstrate modular programming concepts.

## Files

### `math_module.py`
A simple module for mathematical operations.
- **Functions:**
  - `add(a, b)`: Returns sum of two numbers.
  - `subtract(a, b)`: Returns difference.
  - `multiply(a, b)`: Returns product.
  - `divide(a, b)`: Returns division (handles zero division).
  - `calculate_stats(numbers)`: Returns dictionary with sum, max, min, and average of a list.

### `string_module.py`
A simple module for string manipulations.
- **Functions:**
  - `reverse_text(text)`: Returns reversed string.
  - `to_uppercase(text)`: Returns uppercase string.
  - `count_chars(text)`: Returns length of string.
